PWM (Pulse Width Modulation) generator for power management and voltage regulation.

Inputs:
- clk: system clock
- rst_n: active-low asynchronous reset
- enable: PWM enable (0=disabled, counter held at 0, output low)
- period[7:0]: PWM period — counter counts from 0 to period-1, then wraps to 0
- duty[7:0]: duty cycle threshold — pwm_out is high when counter < duty

Outputs:
- pwm_out: PWM output signal
- counter[7:0]: current counter value (visible for verification)

Behavior:
- Reset: counter=0, pwm_out=0
- Disabled (enable=0): counter=0, pwm_out=0
- Enabled: counter increments each clock cycle from 0 to period-1, then wraps to 0
- pwm_out = 1 when counter < duty, pwm_out = 0 when counter >= duty
- If duty=0: output always low (0% duty cycle)
- If duty >= period: output always high (100% duty cycle)
- Changing period or duty takes effect immediately on the next cycle

Verify:
1. Reset clears counter and output
2. Disabled state holds counter at 0 and output low
3. Counter wraps correctly at period-1 back to 0
4. PWM output goes high for exactly 'duty' cycles out of every 'period' cycles
5. 0% duty (duty=0): output stays low
6. 100% duty (duty >= period): output stays high
7. 50% duty: output toggles halfway through period
8. Enable/disable transitions work correctly
